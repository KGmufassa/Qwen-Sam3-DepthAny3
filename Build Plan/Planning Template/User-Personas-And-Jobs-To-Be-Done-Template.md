## 8. Freeze Rule

The persona and JTBD framing is considered frozen when:

- the primary persona is approved
- the main job to be done is approved
- MVP scope aligns with the chosen problem framing

If this framing changes later, reopen:

- PRD
- MVP scope planning
- frontend experience planning
- analytics planning
